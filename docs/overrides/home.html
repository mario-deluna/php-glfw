To create interactive applications, you need more than just GLFW bindings. This extension aims to provide you with all the basic tools necessary to develop interactive, real-time applications in PHP.

  • GLFW Bindings

    The obvious one, from which the name is derived, this library contains almost full support for GLFW, which provides a simple API for creating windows, contexts, and surfaces, as well as for handling input and system events.

  • Cross-Platform

    PHP-GFLW works on Window, MacOS and Linux.
