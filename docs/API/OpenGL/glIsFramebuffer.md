# glIsFramebuffer
determine if a name corresponds to a framebuffer object

```php
function glIsFramebuffer(int $framebuffer) : bool
```

arguments

:    1. `int` `$framebuffer` Specifies a value that may be the name of a
    framebuffer object.

returns

:    `bool`
