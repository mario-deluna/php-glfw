# glMinSampleShading
specifies minimum rate at which sample shading takes place

```php
function glMinSampleShading(float $value) : void
```

arguments

:    1. `float` `$value` Specifies the rate at which samples are shaded within
    each covered pixel.

returns

:    `void`
