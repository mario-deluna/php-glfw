# glGenFramebuffers
generate framebuffer object names

```php
function glGenFramebuffers(int $n, ?int &...$framebuffers) : void
```

arguments

:    1. `int` `$n` Specifies the number of framebuffer object names to generate.
    2. `?int` `$framebuffers` 

returns

:    `void`
